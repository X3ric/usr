##  DotFiles

#### Bin
`ytm-dl`: a command to use [youtube-dl](https://github.com/ytdl-org/youtube-dl) to download MP3 files with thumbnails for [MPD](https://github.com/MusicPlayerDaemon/MPD) (Music Player Daemon).

#### Theme
- GTK theme: `darkarch` based on [Adwaita Amoled](https://www.gnome-look.org/p/1553851)
- Cursor: [`cz-Hickson-Black`](https://www.gnome-look.org/p/1503665) 
- Icons: [`Papirus`](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme)

#### Zsh alias commands
- `ex` command to extract files
- `start` command to start the wm

#### Zsh replaced commands
- `ls` command replaced with lsd 
- `rm/rmdir` commands replaced with rmtrash
